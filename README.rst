Installation
------------

Copy the product to "src/" folder in your instalation;

Install bbbs.blog by adding it to your buildout::

    [buildout]

    ...

    eggs =
        bbbs.blog

    ...
    develop =
        src/bbbs.blog


and then running ``bin/buildout``
